It looks like Ambrosia is having some maintenance issues at the moment. Sorry about that! The project was made in an emulation of an original Game Boy Advance, so it doesn't always play nice with modern web architecture. I'm doing everything I can to get it running in browser. If you would like, you can still play it the traditional way, but it is a little more complicated.
First, click here to download the ROM.
After that, go to gba.ninja and run the file. Pay attention to your keybindings, as all tutorials will be given for GameBoy buttons, not a keyboard.
Oh! GBA Ninja also doesn't like homemade ROMs, and will give you an error message. Just click "Run it anyway", it will work.
